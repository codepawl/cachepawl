\begin{abstract}
Hybrid Mamba-Transformer language models combine attention with State Space Models (SSMs), creating asymmetric memory pressure between linearly scaling Key-Value (KV) blocks and fixed-size SSM states. Existing allocator designs for hybrid models typically either pad SSM state into KV-like pages or statically partition memory, which can waste capacity or fail to adapt under shifting workload distributions. Unified pools pad SSM states to match KV pages, causing severe capacity overestimation, while static dual-pool partitions cannot adapt to shifting prompt distributions. We present Asymmetric Virtual Memory Paging (AVMP), a memory allocator that decouples virtual address spaces across physically heterogeneous pools and dynamically rebalances capacity at runtime. We evaluate AVMP using a strictly deterministic 270-cell sweep across three synthetic workloads. The dynamic AVMP allocator records goodput improvements ranging from 1.83$\times$ to 13.3$\times$ across three synthetic workloads and a 7.6\% cross-workload Out-of-Memory (OOM) reduction compared to the best static baseline. Our logical operation counter guarantees byte-identical reproducibility across all reruns. We scope this design study as a pure Python prototype evaluated on a single GPU.
\end{abstract}
