## Claim Boundary

Supported in this package:

- planner-level over-reservation in the bounded Path C matrix;
- planner-only evidence that `cachepawl-avmp` stays closer to useful bytes than
  the padded planner baseline in the committed RTX 3060 pack;
- advisory artifact-input diagnosis;
- read-only, non-mutating observation.

Not supported in this package:

- runtime allocator replacement;
- runtime cache substitution;
- live VRAM reduction;
- throughput, latency, quality, or accuracy improvement;
- controlled substitution readiness before Mamba state-index and state tensor
  contracts are resolved.
